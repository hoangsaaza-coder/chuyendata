APP XUẤT DỮ LIỆU ĐƠN HÀNG TỪ ẢNH - NỆM VIỆT NHẬT

Cách chạy trên PC Windows:

1. Cài Python nếu máy chưa có:
   - Vào python.org tải Python bản mới nhất.
   - Khi cài nhớ tick ô "Add Python to PATH".

2. Giải nén file ZIP này ra Desktop hoặc thư mục dễ tìm.

3. Bấm đúp file install.bat để cài thư viện lần đầu.

4. Bấm đúp file run_app.bat để mở app.
   - App sẽ chạy trên trình duyệt, thường ở địa chỉ http://localhost:8501

5. Nhập OpenAI API key ở thanh bên trái.

6. Upload ảnh hóa đơn, bấm "Đọc dữ liệu từ ảnh".

7. Kiểm tra lại bảng, chỉnh tay nếu cần.

8. Bấm "Tải file CSV" hoặc copy nội dung để dán vào Google Sheet.

Các trường app xuất:
KHÁCH HÀNG | Tt | SĐT | ĐỊA CHỈ | SẢN PHẨM | KÍCH THƯỚC | SL | TỔNG ĐƠN | GHI CHÚ | NGUỒN ĐƠN | NGƯỜI BÁN

Ghi chú:
- App cần internet để gọi AI đọc ảnh.
- App không lưu API key lên server riêng; key chỉ nhập trong phiên chạy app trên máy của bạn.
- File CSV có thể mở bằng Excel hoặc import vào Google Sheet.
